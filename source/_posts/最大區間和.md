# 最大區間和
>##### 給你$a_1,a_2...,a_n$求出$a_l+...+a_r$最大為何?
<!--more-->
## 思考
我們先定義$dp[i]$代表**以$a[i]$為結尾的最大的區間和**，接著思考初始值$dp[0]$(沒有數字的時候)，毫無疑問是$0$。

根據上面的假設，我們可以得出兩種情況，$dp[i-1]$和$dp[i]$的開頭是一樣的(思考一下一開始的定義)，或是$dp[i]$是一個新的開始，我們選出這兩個之中較大的那個，他就是$dp[i]$的答案。

最後我們只要找出以$a_1,a_2...,a_n$為結尾的區間中最大的那個就可以得到答案。
## 整理&列轉移式
* $dp[i]$代表以$a[i]$為結尾的最大的區間和
* $dp[0]=0$
* 開頭一樣時$dp[i]=dp[i-1]+a[i]$
* 新的開始代表$dp[i]=a[i]$

**轉移式:**
$$ dp[i]=max(dp[i-1]+a[i], a[i]) $$
狀態數$O(n)$, 轉移複雜度$O(1)$, 總複雜度為$O(n)$
## 實際例子
假設$a[\ ]=10,-5,10,1,-3,2,-3$

**Step3**: 這題的答案跟其他$dp$稍微不一樣，要找出$dp[1],...,dp[n]$中最大的才是答案，因為我們$dp[i]$的定義是以$i$為結尾的最大區間和，但是$dp[n]$的區間和不一定會是整個數列中最大的區間和，所以這個例子的答案是16。
## 滾動
從前面的操作可以發現，每次新增一個狀態只會用到前一個狀態，因此，我們其實不需要用到一整條陣列，我們只需要用兩個變數來存會用到的狀態就好(我通常會用$pre$代表前一個狀態，$cur$代表當前狀態)，因為蠻直覺的就不實際操作一遍了，還是不懂可以看看底下的程式碼。

    ```cpp=
#include <algorithm>
#include <iostream>
using namespace std;
#define ll long long int
#define oo 1000000005
int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);
    int t;
    cin >> t;
    while(t){
        t--;
        int n, dp[105], ans = -oo;
        cin >> n;
        int a[n+5];
        for (int i = 1; i <= n;i++){
            cin >> a[i];
        }
        for (int i = 0; i <= n;i++){
            dp[i] = 0;
        }
        for (int i = 1; i <= n; i++){
            dp[i] = max(dp[i - 1] + a[i], a[i]);
            ans = max(ans, dp[i]);
        }
        cout << ans <<"\n";
    }
    return 0;
}
```

    ```cpp=
#include <algorithm>
#include <iostream>
using namespace std;
#define ll long long int
#define oo 1000000005
int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);
    int t;
    cin >> t;
    while(t){
        t--;
        int n, ans = -oo, pre=0, cur=0;
        cin >> n;
        int a[n+5];
        for (int i = 1; i <= n;i++){
            cin >> a[i];
        }
        for (int i = 1; i <= n; i++){
            cur = max(pre + a[i], a[i]);
            ans = max(ans, cur);
            pre = cur;
        }
        cout << ans <<"\n";
    }
    return 0;
}
```
