# 0/1背包問題

>##### 給定背包可以承受的最大重量$W$以及$n$個物品，第$i$個物品的重量為$w[i]$，第$i$個物品的價值為$p[i]$，我們要選物品放進背包(<font color=#FF0000>只能選或不選</font>)，求最大價值為何?

<!--more-->

## 思考
我們先定義$dp[i][j]$為**前$i$個物品中背包容量為$j$時可裝的最大價值**，再來思考初始值$dp[i][0]$(背包容量為0的價值)和$dp[0][j]$(不取東西的最大價值)，我們可以發現當我們不拿東西或是背包容量為0時最大價值都為0。

接著分析選擇第$i$個物品所造成的影響，如果我們不選，那麼我們的價值就會跟前$i-1$個物品所提供的價值一樣，也就是$dp[i-1][j]$，如果我們選，那麼價值就會是當前背包容量減第$i$個物品的重量時前$i-1$個物品可裝入的最大價值再加上第$i$個物品的價值，也就是$dp[i-1][j-w[i]]+p[i]$

最後我們分析甚麼時候可以選取第$i$個物品，如果當前背包容量比第$i$個物品的重量還小，那我們就不能選，如果當前背包容量比第$i$個物品的重量還大，那我們就可以依照選或不選之後誰提供的價值比較大作為當前的最大價值。
## 整理&列轉移式
* $dp[i][j]$為前$i$個物品中背包容量為$j$時可裝的最大價值
* $dp[i][0]=0$和$dp[0][j]=0$
* $j>=w[i]$時$dp[i][j]=max(dp[i-1][j], dp[i-1][j-w[i]]+p[i])$
* $j<w[i]$時$dp[i][j]=dp[i-1][j]$

**轉移式:**
$$ dp[i][j]=\begin{cases}	
if(j<w[i])& \text{$dp[i-1][j]$}\\\\
if(j>=w[i])&\text{$max(dp[i-1][j], dp[i-1][j-w[i]]+p[i])$}
\end{cases} $$

狀態數$O(nW)$, 轉移複雜度$O(1)$總時間複雜度為$O(nW)$

## 滾動
從前面的方法我們可以發現，我們每次只用到$dp[i-1][j]$和$dp[i-1][j-w[i]]$以及$dp[i][j]$這三格，也就是說，$i$的部分我們只會用到兩個，所以我們可以使用一條一維陣列實作，而轉移式為:
$$ dp[j]=\begin{cases}	
if(j<w[i])& \text{$dp[j]$}\\\\
if(j\geq w[i])&\text{$max(dp[j], dp[j-w[i]]+p[i])$}
\end{cases} $$

    #include <algorithm>
    #include <iostream>
    using namespace std;
    #define ll long long int
    #define oo 1000000005
    int dp[105][10005]={0};
    int main()
    {
        ios::sync_with_stdio(0);
        cin.tie(0);
        int n;
        while(cin>>n){
            int w[n + 5], p[n + 5], W;
            for (int i = 1; i <= n;i++){
                cin >> w[i] >> p[i];
            }
            cin >> W;
            for (int i = 0; i <= n;i++){
                dp[i][0] = 0;
                
            }
            for (int j = 0; j <= W;j++){
                dp[0][j] = 0;
            }
            for (int i = 1; i <= n; i++)
            {
                for (int j = W; j >= 1; j--)
                {
                    if (j >= w[i])
                    {
                        dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - w[i]] + p[i]);
                    }
                    else{
                        dp[i][j] = dp[i - 1][j];
                    }
                }
            }
            cout << dp[n][W]<<"\n";
        }
        return 0;
    }
