# 最長共同子序列(Longest Common Sequence)

>##### 一個序列的子序列是指在不破壞原始序列順序的情況下取出的任意長度的序列，假設有一個字串$s=zxcvbnm$，而$xcvb$、$zxcvbnm$、$z$、$zcnm$之類的都是$s$的子序列。
>##### 這題的題目是給兩個字串$s$長度為$n$和$t$長度為$m$,求出他們的最長共同子序列($LCS$)。
<!--more-->
## 思考
我們先定義$dp[i][j]$為**字串$s$的前$i$個字元和字串$t$的前$j$個字元的$LCS$** ，然後思考初始值$dp[i][0]$和$dp[0][j]$，他們分別是$s$沒有字元或是$t$沒有字元的情況，顯而易見的，這種情況下$LCS$一定等於$0$。

再來，我們分析一下$s[i]=t[j]$時會發生甚麼事，這表示$s[i]$和$t[j]$一定會成為$LCS$的一分子，所以可以把$s[i]$和$t[j]$都刪除並找到他們的$LCS$再加一，而$s[i] \neq t[j]$時代表$s[i]$或$t[j]$中有一個是沒用的，因此我們可以看看刪除$s[i]$還是$t[j]$的效果會比較好。

最後，$dp[n][m]$就是我們的答案，實作時要注意我們的$dp$是從$1$開始，而字串的比對要從$0$開始，所以<font color=#FF0000>實作時字串部分的$i$和$j$記得減一</font>。
## 整理&列轉移式
* $dp[i][j]$為**字串$s$的前$i$個字元和字串$t$的前$j$個字元的$LCS$**
* $dp[i][0]=0$和$dp[0][j]=0$
* $s[i]=t[j]$時$dp[i][j]=dp[i-1][j-1]+1$
* $s[i] \neq t[j]$時$dp[i][j]=max(dp[i][j-1], dp[i-1][j])$
* 實作時字串部分的$i$和$j$記得減一

**轉移式**
$$ dp[i][j]=\begin{cases}	
if(s[i]=t[j])& \text{$dp[i-1][j-1]+1$}\\\\
if(s[i] \neq t[j])&\text{$max(dp[i][j-1], dp[i-1][j])$}
\end{cases} $$

狀態數$O(nm)$, 轉移複雜度$O(1)$, 總時間複雜度$O(nm)$

## 滾動
從前面的轉移式我們可以發現，我們只會用到$i-1$和$i$的狀態，也就是說其實我們只需要用到兩列就夠了，我們把第$0$列當作第$i-1$列，第$1$列當作第$i$列，再把第$1$列複製到第$0$列，一直到結束，不過這這樣會多出複製的時間。改良的方式是用輪流的，先從第$1$列開始算，再算到第$0$列，再回到第$1$列...不斷重複直到結束，通常使用取餘數的方法來做，做法可以參考下方程式碼。

```cpp=
#include <algorithm>
#include <iostream>
using namespace std;
#define ll long long int
#define oo 1000000005
int dp[1005][1005];
int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);
    string s, t;
    while(cin >> s){
        cin >> t;
        int n = s.length();
        int m = t.length();
        for (int i = 0; i <= n;i++)
            dp[i][0] = 0;
        for (int j = 0; j <= m;j++)
            dp[0][j] = 0;
        for (int i = 1; i <= n;i++){
            for (int j = 1; j <= m;j++){
                if(s[i-1] == t[j-1])
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                else
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
        cout << dp[n][m]<<"\n";
    }
    return 0;
}
```
