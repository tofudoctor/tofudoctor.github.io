# 最長遞增子序列(Longest Increasing Subsequence)

>##### 一個序列的子序列是指在不破壞原始序列順序的情況下取出的任意長度的序列，假設有一個字串$s=zxcvbnm$，而$xcvb$、$zxcvbnm$、$z$、$zcnm$之類的都是$s$的子序列。
>##### 這題的題目是給$a_1,a_2,...,a_n$，求出$a_l,...,a_r$，其中$a_l$~$a_r$是遞增的，也就是說求出題目的最長遞增子序列($LIS$)
<!--more-->
## 思考
我們先定義$dp[i]$為**以$a[i]$為結尾的$LIS$長度**，然後設定初始值$dp[1]=1$，因為$a_1$一定只能形成長度為$1$的$LIS$。

接著，分析一下甚麼時候可以把$a[i]$加進$LIS$，條件就是要比原本$LIS$的結尾大，但$dp[i]$會是多少呢?我們可以發現，符合上述條件的值不一定只有一個，因此，我們需要把$a[i]$前面的值都檢查一遍，找出$LIS$長度最大的那個再加一就會是當前的$dp[i]$。

最後，因為我們$dp[i]$的定義是以$a[i]$為結尾的$LIS$長度，因此不保證最後一個一定就是最大的，所以我們需要把$dp[1]$~$dp[n]$都檢查一遍，最大的就是我們的答案。
## 整理&列轉移式
* $dp[i]$為**以$a[i]$為結尾的$LIS$長度**
* $dp[1]=1$
* 把$a[i]$前面$<a[i]$的值都檢查一遍，找出$LIS$長度最大的那個再加一就會是當前的$dp[i]$
* 做完記得把$dp[1]$~$dp[n]$都檢查一遍，最大的才是我們的答案

**轉移式:**
$$ dp[i]=max(dp[j]+1)_{j<i\ \\&\ a[j]<a[i]} $$

狀態數$O(n)$, 轉移複雜度$O(n)$, 總複雜度為$O(n^2)$
## 實際例子
假設$a[\ ]=1,5,3,7,4$
**Step1**: 設初始值
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  <font color=#FF0000>1</font>   |     |     |     |     |

**Step2**: 填表格
找出$dp[1]$~$dp[1]$之中最大且$a[j]<5(a[2])$的，取$1(dp[1])+1$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  <font color=#FF0000>2</font>   |     |     |     |

找出$dp[1]$~$dp[2]$之中最大且$a[j]<3(a[3])$的，取$1(dp[2])+1$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  2  |  <font color=#FF0000>2</font>   |     |     |

找出$dp[1]$~$dp[3]$之中最大且$a[j]<7(a[4])$的，取$2(dp[2])+1$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  2  |  2  |  <font color=#FF0000>3</font>   |     |

找出$dp[1]$~$dp[4]$之中最大且$a[j]<4(a[5])$的，取$2(dp[3])+1$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  2  |  2  |  3  |  <font color=#FF0000>3</font>   |

**Step3**: 這題的答案跟其他$dp$稍微不一樣，要找出$dp[1],...,dp[n]$中最大的才是答案，因為我們$dp[i]$的定義是以$i$為結尾的$LIS$，他不一定會最長，因此我們需要把$dp[\ ]$的結果跑一遍找出最大值，這個例子的答案是$3$。
## 優化
不過仔細想想，好像有些東西用不到?假設我們正在找$dp[i]$，當$a[j]$比$a[i]$小而且$LIS$還比$a[i]$的大時，$dp[i]$可以說是不可能會被取用，因為能接在$a[i]$後面的一定能接在$a[j]$後面，而$a[j]$的效果還會比$a[i]$好，也就是說在相同長度的情況下$a[i]$越小被放進$LIS$的機會就越大。

像是$5$被放進$LIS$的機會比$4$小，因為$5>4$且$2<3$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  4  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  <font color=#FF0000>2</font>  |  2  |  3  |  <font color=#FF0000>3</font>   |

因此，我們的$dp[i]$有了新的定義，就是**長度為$i$時的最小結尾**，而我們要做的就是找出$a[i]$有沒有比$dp[\ ]$內的值小，如果有就代表$a[i]$成為$LIS$的潛力比原本的值還大，那我們就找出$dp[\ ]$中$>=a[i]$的第一個數替換成$a[i]$，如果沒有，代表$a[i]$可以接在當前的$LIS$後面，直接把$a[i]$加到$dp[\ ]$的最後面就好了，而$LIS$的長度非常簡單，看定義就知道，$dp[\ ]$長度多少答案就是多少。
### 整理
* $dp[i]$是**長度為$i$時的最小結尾**
* 如果$a[i]>dp[\ ]$中所有的值，加進$dp[\ ]$的結尾
* 否則，找出$dp[\ ]$中第一個$>=a[i]$的數替換成$a[i]$
* <font color=#FF0000>可以用二分搜加速</font>，原因底下說明
* $LIS$的長度$=dp[\ ]$長度

### 實際例子
找不到比$>=1$的，直接加入$1$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  2  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  <font color=#FF0000>1</font>   |     |     |     |     |

找不到比$>=5$的，直接加入$5$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  2  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  <font color=#FF0000>5</font>  |     |     |     |

第一個$>=3$的數是$5$，將其替換成$3$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  2  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  |  <font color=#FF0000>3</font>  |     |     |     |

找不到比$>=7$的，直接加入$7$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  2  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  | 3   |  <font color=#FF0000>7</font>   |     |     |

第一個$>=2$的數是$3$，將其替換成$2$
|              $a[\ ]$               |  1  |  5  |  3  |  7  |  2  |
|:----------------------------:|:---:|:---:|:---:|:---:|:---:|
| $dp[\ ]$ |  1  | <font color=#FF0000>2</font>  |  7   |     |     |

最後$dp[\ ]$的長度是$3$，因此本題答案為$3$。而可以用二分搜的原因是因為$dp[\ ]$的值都是遞增的，因此可以用二分搜加速，加速後的時間複雜度為$O(n\log n)$

```cpp=
#include <algorithm>
#include <iostream>
#include <string.h>
#include <vector>
using namespace std;
#define ll long long int
#define oo 1000000005
int main()
{
    ios::sync_with_stdio(0);
    cin.tie(0);
    vector<int> a;
    vector<int> pos;
    vector<int> dp;
    vector<int> ans;
    int n;
    while(cin>>n){
        a.push_back(n);
        pos.push_back(0);
    }
    int l = a.size();
    for (int i = 0; i < l;i++){
        auto t = lower_bound(dp.begin(), dp.end(), a[i]);
        //vector<int>::iterator t = upper_bound(dp.begin(), dp.end(), a[i]);
        if(t == dp.end()){
            dp.push_back(a[i]);
            pos[i] = dp.size();
        }
        else{
            dp[t - dp.begin()] = a[i];
            pos[i] = t - dp.begin()+1;
        }
        
    }
    int cur_pos = dp.size();

    for (int i = pos.size()-1; i >= 0;i--){
        if(pos[i] == cur_pos){
            ans.push_back(a[i]);
            cur_pos--;
        }
    }
    cout << dp.size() << "\n";
    cout << "-\n";
    for (int i = dp.size() - 1; i >= 0;i--){
        cout << ans[i] << "\n";
    }
    return 0;
}
```
